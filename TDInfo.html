To-Do List App

DEMO

HOW IT WORKS

The idea for this app was to let users list down as many tasks as they needed to complete.
They are also able to cross off tasks as they complete them or
completely delte them if needed. One of the big benifits of my app is
that it saves your changes into the browser so you don't have to worry about your tasks being delted if you leave and come back.

THE IDEA

The program awaits a user to input text into the add task input bar, it will send out an alert if the button is clicked and their is no
text. If their is content when the button is clicked it uses a function to create a new list item in the html and fill it will the user input.
The cross out button works soley with css to create a red line through the text area of the list. The X button works by removing the list item and then updating the local storage to keep it
removed. It saves the data by writing it into the local storage and then calling this function after every button click so every action is saved.
